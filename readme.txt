Этот синтаксис предназначен для работы с файлами скриптов
Установка:
в Notepad++ нжимаете "Синтаксисы" - "Польз. определение синтаксиса" - "Задать свой синтаксис" - "Сохранить как" (сразу создаете синтаксис)
В открывшемся окне нажимаете "Импорт" и выбираете скачанный синтаксис (Не забудьте распаковать архив)
Данный синтаксис не предназначен для работы с файлами .gui и .gfx
